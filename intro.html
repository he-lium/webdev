HTML

Hyper-text Markup Language (HTML) is a language that describes the structure and layout of a web page. A website written in HTML is made up of elements that a web browser reads and uses to display the website properly.

A sample website
            
<!DOCTYPE html>
<html>
    <head>
        <title>Page Title</title>
    </head>
    <body>

        <h1>My First Heading</h1>
        <h2>My Second Heading</h2>
        <p>My first paragraph.</p>

    </body>
</html>
HTML Elements

A HTML element is made up of a start tag and an end tag, with the content of the element being within those tags. The content can be text, or even more elements, or be empty.

<tagname>Content goes here...</tagname>
<!DOCTYPE html> declaration

This DOCTYPE line is at the top of each HTML page. It tells the web browser that this is a HTML5 document.

<html> tag

This is the root element of the HTML page. All the info and content in the document is within the html tags.
HTML documents usually contain two main elements: the <head> tag and the <body> tag.

<head>
The head tag is for storing descriptions and general info of the webpage. One of the 1elements that belongs in here is the <title> tag, which displays the website title in the tab bar. 
<head>
    <title>My website</title>
</head>
<body>

The <body> element is the one containing all the content of the HTML document, such as text, links, images, tables, etc. All the elements placed in the body will be displayed by the web browser for the user to see.

Comments

Comments aren't displayed to the user, but are there for you to help add documentation to your website. i.e. explain a section of code, write down ideas and things that need to be done.

Page breaks

<br> is used to represent page breaks. They add a new line to split the content, rather than creating a new paragraph. It's used a lot for things like poems. Note that it does not need a closing tag, i.e. don't do </br>

Images

The <img> tag is used to display images in HTML.
The src attribute specifies the URL or location of the image, and the alt attribute provides alternate text for the image if the user somehow cannot read it (slow connection, error or user is blind and using a screen reader).

Note that the tag does not require a closing tag, so no </img> is necessary. 

<img src="images/bridge.jpg" alt="Sydney Harbour Bridge">
