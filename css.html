What is CSS?

So far your HTML pages are full of text, headings, links and images. They may look rather dull, since they're the same black Times New Roman font on a white background. Since HTML is for describing the structure of the web page, we need something else to format it.

That's where CSS fits in. CSS (or Cascading Style Sheets) is the language for describing how elements are presented. It adds instructions for the layout, colours and fonts that are displayed in a webpage.

Adding CSS formatting to your websites

There are two ways to add CSS to your HTML pages. We'll show you the simpler two ways of doing this:

1. Inline attribute

CSS can be added inline to HTML tags using the style attribute. This applies the formatting to a single HTML element.

<!-- Note that these styles won't apply to other h1 or p tags, just these -->
<h1 style="font-size: 40px;color: red;">This is a big red heading</h1>
<p style="color: purple">Purple rain, purple rain! (R.I.P Prince :( )</p>

2. Using the <style> tag
Another way of adding style is including the <style> tag in the HTML <head>. For example: 
<head>
    <title>My website</title>
    <!-- add styles to all <h1> tags -->
    <style>
        h1 {
            font-size: 40px;
            color: cyan;
        }

        .intro {
            font-size: 60px;
            color: black;
        }
    </style>
    <!-- add style to tags that have attribute class="intro" -->
</head>
Class selectors

The intro in the sample above is called a class selector. It applies the styles (black colour, 60 pixel font size) to all elements that have their class attribute as "intro". For example, it would apply to this:

<p class="intro">This paragraph is huge!</p>

This paragraph is huge!

Colors

You can use CSS to make your web page a lot more colourful by adding colour to your HTML elements. Colours are made by mixing RED, GREEN and BLUE light.

<h4 style="color: rgb(255, 0, 255);">This heading appears magenta</h4>

This heading appears magenta

When using rgb(r, g, b) to describe a colour like the heading above, you specify three numbers for the red, green and blue components of that number. Each number must be between 0 and 255 inclusive. If you need help finding the right colour, a tool that uses sliders to get the right colour can be found here.

Colour names

An alternative to using numbers to describe the colour, you can also pick from a number of different colour names. For example:

<h4 style="color: darkblue">This heading is dark blue</h4>

This heading is dark blue

Background Colour

Use the background-color property to change the highlighter colour behind the text:

<p style="background-color: rgb(30, 30, 30); color: darkorange">Happy Halloween! Mwahahahahaha</p>

Happy Halloween! Mwahahahahaha

If you want the colours to apply to the entire website, add those properties to the body tag:

<head>
    <!-- Apply this for a hacker green look! -->
    <style>
        body {
            background-color: black;
            color: limegreen;
        }
    </style>
Fonts

You can use CSS to change the font of the text using the font-family property:

.arial {
    font-family: Arial, Helvetica, sans-serif;
    font-size: 20px;
}
<p class="arial"> Arial, unlike Times New Roman, is a sans serif font.</p>

Arial, unlike Times New Roman, is a sans serif font.

Other possible fonts:

font-family: Courier New, Courier, monospace;

Underlining

Use the text-decoration property to add or remove underlines or strikethroughs.

<h4 style="text-align: center; text-decoration: underline">My giant treasure vault :)</h4>
<h4 style="text-align: center; text-decoration: line-through">Your ATAR rank is < 30.00</h4>

My giant treasure vault :)

Your ATAR rank is < 30.00

You know how links by default have an underline? The text-decoration can also remove underlines:

.no-underline {
    text-decoration: none;
}
<p><a href="https://www.unsw.edu.au/" target="_blank" class="no-underline">University of New South Wales</a></p>

University of New South Wales

Text Shadows

The text-shadow property adds shadows to text. When using the property, specify in the following order:

  • The position of the horizontal shadow
  • The position of the vertical shadow
  • The shadow colour

<h6 style="text-shadow: 1px 2px darkblue">"I believe life is an intelligent thing: that things aren't random." -Steve Jobs</h6>
"I believe life is an intelligent thing: that things aren't random." -Steve Jobs
