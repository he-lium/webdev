Lists

Lists make up a large part of everyday life, and are useful for a wide range of applications, including:

  • Driving instructions
  • Names for attendance
  • Recipe ingredients and instructions
  • Your collection of video games
and so on. HTML provides several ways to help display lists in websites.

<ol>

Use the <ol> tag to display ordered lists, where the order is important and numbers are used as markers. The li tag is used to represent the list items.

Styling Lists

You can use CSS and the list-style-type property to change what bullet points the list should use.

<ul style="list-style-type: square">
    <li>Caffeine</li>
    <li>Sugar</li>
    <li>Milk</li>
</ul>
  • Caffeine
  • Sugar
  • Milk

Tables

Tables are used in your website to organise data into columns and rows. HTML provides several elements to form the structure for tables. An example is given below:

<table>
    <caption>Yearly Results Report</caption>
    <tr>
        <th>Subject</th>
        <th>Marks</th>
        <th>Grade</th>
    </tr>
    <tr>
        <td>English</td>
        <td>77</td>
        <td>B</td>
    </tr>
    <tr>
        <td>Mathematics</td>
        <td>91</td>
        <td>A</td>
    </tr>
</table>
Yearly Results Report
Subject Marks Grade
English 77 B
Mathematics 91 A

An explanation of what each element does:

  • <table> – Tells the browser that the data within this tag is to be organised in a table. Contains one or more <tr> rows.
  • <caption> – Table caption that briefly outlines what the table and data are about.
  • <tr> – Table row. Each row has a number of cells (columns).
  • <th> – Table header. Belongs to the first header and the describes the data in its own column.
  • <td> – Table cell (table data). Having multiple cells in a row will make more columns in the table.
Table styling and borders

Adding borders to tables makes it easier for the user to read the table. Applying CSS styles to table borders involves changing the border property in each table cell:

table {
    border-collapse: collapse;
}

th, td {
  border: 1px solid #cecfd5;
  padding: 10px 15px;
}

sidenote: the border-collapse property above prevents adjacent cells from doubling their own borders such that it would be 2px wide.

Embedding YouTube videos

To add YouTube videos to your website:

  1. Look at the video URL and get the video id.
    • If the url is https://www.youtube.com/watch?v=qOOaH6BAwvY then the id is qOOaH6BAwvY
  2. In your HTML document, add this <iframe> tag, including your video id in the src attribute: 
    <!-- Insert the video id after embed/ -->
<iframe src="https://www.youtube.com/embed/qOOaH6BAwvY"
width="800" height="450" frameborder="0">
</iframe>
  3. Specify the height and width of the YouTube embedded video.
